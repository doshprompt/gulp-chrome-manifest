> Gulp plugin for Chrome Manifest


## Install

```sh
$ npm install --save gulp-chrome-manifest
```


## Usage

```js
var gulp-chrome-manifest = require('gulp-chrome-manifest');

gulp-chrome-manifest('Rainbow');
```
